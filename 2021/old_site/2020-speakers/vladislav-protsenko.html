The Talk

Reveal is a Clojure REPL with UI that enables data exploration in a way never seen before: select printed text and evaluate code directly on a value that produced it, build charts only using data, explore object fields in debugger-like inspector, easily create views that matter to you or your domain.

In this talk I'm going to explain what is Reveal, why I made it and how to use it.
