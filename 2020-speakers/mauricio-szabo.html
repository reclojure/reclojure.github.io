Chlorine - a Socket REPL package with some super-powers

Chlorine is a socket-REPL package for the Atom editor - and in the future, others that can run Javascript. What I am going to show is how it is not only a way to connect to Clojure and evaluate code, but also how its incredible customization features can it be useful in data-science, code insights, and multiple other situations now and in the future
