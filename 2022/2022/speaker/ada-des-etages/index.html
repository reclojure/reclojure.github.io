Using rule engines for charts and poetry

My journey through building websites with rule engines. I will discuss a Clojure charting application built using a general-purpose rules engine to introduce the concepts. Then I will walk-through applications based on a Clojurescript rule engine made specifically for rendering React components. The applications include a todo list, a deck of affirmations, and a text editor for writing poems. Finally, I will show you how you can build your own applications using a starter that I'm making.
