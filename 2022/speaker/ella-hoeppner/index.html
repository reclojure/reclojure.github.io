Vlojure - A new way to write ClojureScript

By Ella Hoeppner

Vlojure is a new interface for programming ClojureScript which lets users view and edit code using a visual, drag-and-drop based interface, rather than a traditional, text-based interface. Vlojure is an open-source project and is in an early stage at the moment, but I hope to continue expanding Vlojure, and I'm especially interested in exploring the possibility of using it as a teaching tool for new programmers. You can try Vlojure right now in your browser at vlojure.io
